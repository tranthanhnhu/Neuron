# NuSMV (local install)

Windows binary **NuSMV 2.6.0** is unpacked under `nusmv/NuSMV-2.6.0-win64/bin/NuSMV.exe`.

The CLI (`snn_mc`) auto-detects this path when NuSMV is not on system PATH.

Manual run:

```powershell
& "tools\nusmv\NuSMV-2.6.0-win64\bin\NuSMV.exe" -h
```

Re-download (if missing): from [NuSMV downloads](https://nusmv.fbk.eu/distrib/NuSMV-2.6.0-win64.tar.gz), extract into `tools/nusmv/`.
